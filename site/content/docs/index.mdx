**Macaron Artifacts** publishes the plugin manifests, local WebUI runtime, GenUI tooling, and docs for running Macaron with Claude Code, Codex, and Kimi Code.

## What You Get

<Cards>
  <Card title="Visual Sessions" href="/docs/usage">
    Browse workspaces and sessions with previews, then continue a turn from the browser.
  </Card>
  <Card title="Live Chat" href="/docs/usage">
    Stream thinking, tool calls, and GenUI previews from supported agent runtimes.
  </Card>
  <Card title="Provider Controls" href="/docs/usage">
    Run against an ambient login or a compatible endpoint such as Macaron, OpenRouter, or LiteLLM.
  </Card>
  <Card title="genui-builder Skill" href="/docs/usage">
    The bundled skill lets supported agents produce GenUI TSX from the command line.
  </Card>
</Cards>

Head to [Usage](/docs/usage) to install and open the WebUI, or dig into the [Architecture](/docs/architecture/overview) to see how it all fits together.
